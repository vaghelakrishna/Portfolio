## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd react-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Pages

- **Home** (`/`) - Main landing page with about, work, and experience sections
- **Projects** (`/projects`) - Showcase of all projects with detailed cards
- **Chat** (`/chat`) - Interactive chat interface for communication

## Animations

The portfolio features various animations powered by Framer Motion:

- Page transitions
- Staggered animations for lists
- Hover effects
- Custom cursor tracking
- Typing indicators in chat
- Smooth scrolling

## Customization

### Colors
The color scheme can be modified in `tailwind.config.js` or by updating the CSS classes throughout the components.

### Content
Update the content in each page component:
- Personal information in `Home.jsx`
- Project details in `Projects.jsx`
- Chat responses in `Chat.jsx`

### Assets
Replace images in the `public/assets/` directory with your own.
